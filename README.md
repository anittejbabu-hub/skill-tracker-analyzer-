The Skill Tracker Analyzer is a simple project designed to help users track and analyze their skill development over time. The system records different skills, progress levels, and learning activities, allowing users to monitor their improvement.

This project analyzes the stored skill data and provides insights about strengths, weaknesses, and progress trends. It helps learners understand which skills are improving and which require more practice.

The project was developed using Visual Studio Code as the development environment.

🎯 Objectives
Track user skills and learning progress
Analyze skill performance and improvement
Identify strong and weak skill areas
Provide simple insights for better learning planning
🛠️ Technologies Used
HTML – Structure of the application
CSS – Styling and layout
JavaScript – Basic functionality and data handling
Visual Studio Code – Development environment
✨ Features
Add and manage different skills
Track progress for each skill
Analyze skill development over time
Simple and user-friendly interface
